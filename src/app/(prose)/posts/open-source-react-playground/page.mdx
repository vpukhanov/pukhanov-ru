# Open-source React Playground

Я задеплоил простейший React Playground на [react.pukhanov.ru](https://react.pukhanov.ru). Его удобно использовать для код-ревью, и в целом для того чтобы продемонстрировать что-то на примере, с фокусом на логике. Прямого редактора CSS там нет, редактируется JSX.

Исходники можно посмотреть [в моем GitLab](https://git.pukhanov.ru/vyacheslav/react-playground). Там нет документации, но это достаточно простой и прямолинейный проект на Next.js: `npm install` и `npm run dev` запустят его локально.

Код, введенный в редактор playground'a, бандлится прямо на клиенте с помощью WASM-сборки SWC, поэтому на сервер ничего не отправляется и нигде не хранится.
